# Phase 2 — くずし字OCR コスト試算と検証ログ

OCR は **Claude API の vision** を使用（既定モデル `claude-sonnet-4-6`、
精度が要る箇所のみ上位モデル `claude-opus-4-8` を検討）。

## 料金（2026-06 時点）

| モデル | 入力 $/1M tok | 出力 $/1M tok |
|--------|--------------|--------------|
| `claude-sonnet-4-6`（既定） | $3.00 | $15.00 |
| `claude-opus-4-8`（高精度） | $5.00 | $25.00 |

Batches API を使うと **50%** で処理可能（即時性不要のため全件展開時に有力）。
プロンプトキャッシュで翻刻指示部の入力コストはさらに圧縮可能。

## 1 ページあたりの概算（暫定・要実測）

画像トークンは概ね `幅×高さ / 750`。`L`（1200×869）≈ **約1,390 tok**。

`claude-sonnet-4-6` 即時API・`L` 画像・密な本文ページ想定:

| 項目 | トークン | 単価 | 小計 |
|------|---------|------|------|
| 入力（画像 ~1,390 + 翻刻指示 ~400） | ~1,800 | $3/1M | ~$0.0054 |
| 出力（翻刻 ~600字 ≈ ~1,000 tok） | ~1,000 | $15/1M | ~$0.015 |
| **合計（1ページ）** | | | **≈ $0.020** |

Batches 適用で **≈ $0.010/ページ**。

> 上記は机上値。**華頂要略の数ページで実測し、本表を更新してから全件展開を判断する。**

## 検証ログ（実測 2026-06-22）

華頂要略 首巻の目次4見開き（media 0000093233–0000093236）を `claude-sonnet-4-6` で翻刻。

| media | 解像度 | 確信度 | □ | 入力tok | 出力tok | 費用 |
|-------|-------|-------|---|--------|--------|------|
| 0000093233 | O | 0.88 | 0 | 2,385 | 251 | $0.0224 |
| 0000093234 | O | 0.82 | 0 | 2,385 | 321 | $0.0238 |
| 0000093235 | O | 0.72 | 0 | 2,385 | 335 | $0.0235 |
| 0000093236 | O | 0.72 | 1 | 2,385 | 325 | $0.0234 |
| **合計** | | | | | | **$0.0931**（平均 **$0.0233/見開き**） |

精度: 門首御伝の歴代門跡目次として概ね妥当に翻刻（例: 第十七入道尊祐親王／自元祿十二年
至享保十六年 等が画像と一致）。系統的な誤りとして「御→柳」「弟→苔」の取り違えが散見
（"門首御伝"→"門首柳伝" 等）。校正で容易に直る水準。

### 重要な発見: 解像度エスカレーションがコストを倍増させる

- 上表は **4見開きすべてが L→O にエスカレート**した結果（各 L呼び出し＋O呼び出しで2回課金）。
- 既定しきい値（確信度<0.85 または □≥8）が目次のような版面でも発動し、**Oでも確信度は
  0.72–0.88 のまま**＝エスカレーションは精度をほぼ改善せず費用だけ倍化。
- 注: `claude-sonnet-4-6` は長辺1568pxに自動縮小するため、O(原寸3500px)はモデルには
  1568px、L(1200px)はそのまま渡る。Oは入力2,385tok、Lは約1,390tok。

**改訂方針（推奨）**: 既定を **L のみ**運用とし、エスカレーションは
「確信度<0.60 かつ □が一定数以上」など真に判読困難なページに限定する。
これで 1見開き **≈ $0.010**（L単独）に下がる。Batches でさらに半額。

### 改訂コスト試算

| 運用 | 1見開き | 華頂要略(17,020見開き) |
|------|--------|----------------------|
| L単独・即時API | ~$0.010 | ~$170 |
| L単独・Batches | ~$0.005 | ~$85 |
| L→O常時(現状) ・即時 | ~$0.023 | ~$397 |

カタログ全件（2,716資料・総見開き数は Phase 1 で確定。仮に10万〜30万見開き）:
- **L単独 Batches: 約 $500〜$1,500** が現実的な目安。判読困難ページのみ O 再処理。
- 翻刻不要ページ（白紙・表紙・奥付・カラーチャート）をスキップすればさらに減。
